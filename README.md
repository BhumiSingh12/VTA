# V-TRACK Pro – Real-Time Vehicle Tracking Dashboard

A **beautiful, modern, fully functional** real-time vehicle tracking system built with **React + Vite + Tailwind CSS + Leaflet + Chart.js**

Live Demo: (https://vta-67.netlify.app/) (or run locally)

---

### Features
- Real-time moving vehicles on India map (Leaflet)
- Live speed updates & overspeed alerts
- Watchlist detection with instant notifications
- Interactive vehicle details modal
- Stunning **Dark + Hot Pink & Purple** theme
- Animated glowing UI with glassmorphism cards
- Real-time speed trend chart
- Toast notifications
- Pause / Resume simulation
- Responsive design (mobile + desktop)

---

### Tech Stack
- **React 18** + **Vite** (blazing fast)
- **Tailwind CSS** (custom dark + pink theme)
- **Leaflet** (interactive map)
- **Chart.js** + **react-chartjs-2** (speed trends)
- **React-Leaflet** (React + Leaflet integration)
- Pure JavaScript simulation (no backend needed)

---
